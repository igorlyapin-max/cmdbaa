# Карта регистрации событий

| ID | Событие | Уровень | Назначение |
|---|---|---|---|
| LOG-01 | Проверка CMDBuild session | security/info | Аудит доступа к custom page |
| LOG-02 | Ошибка CMDBuild session | security/warn | Выявление невалидной сессии |
| LOG-03 | Загрузка и разбор VSDX | info | Трассировка подготовки шаблона |
| LOG-04 | Результат checksum | info/warn | Контроль целостности файла |
| LOG-05 | Создание версии контракта | audit/info | Аудит изменения правил конвертации |
| LOG-06 | Сохранение enriched VSDX | audit/info | Аудит выпуска шаблона |
| LOG-07 | Построение плана объектов | audit/info | Аудит подготовки создания |
| LOG-08 | Создание объекта CMDBuild | audit/info/error | Аудит записи в CMDB |
| LOG-09 | Ошибка валидатора CMDB | warn/error | Диагностика бизнес-валидации |

События должны содержать пользователя CMDBuild, код контракта, код версии
контракта, имя файла и идентификатор операции, если эти данные доступны.
