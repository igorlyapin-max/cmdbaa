# CMDB BAA: архитектурные артефакты

Артефакты подготовлены по требованиям `../aa.txt`. Для текущего репозитория они
оформлены как Markdown-схемы и таблицы. При подготовке формального пакета их
можно перенести в VSDX или изображения, сохранив идентификаторы потоков и
названия систем.
